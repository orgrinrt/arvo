## `mock/crates/arvo-bits/src/bitfield.rs`

### Extend the `bitfield!` macro with per-field `<FIELD_UPPER>_MASK`

For each sub-range `$field: $field_bits at $lo`, generate:

```rust
pub const ${concat($field, _MASK)}: $crate::Bits<$n, $crate::Hot> = {
    let mask: u64 =
        if $field_bits == 64 { u64::MAX }
        else { (1u64 << $field_bits) - 1 };
    $crate::Bits::<$n, $crate::Hot>::new(mask << $lo)
};
```

Placed inside the generated `impl $name { ... }` block alongside
the existing `BITS`, `new`, `from_bits`, `to_bits`, accessors, and
setters. The `${concat(...)}` metavar expression upper-cases the
field name (already required by the existing setter-name generator
at this macro site).

Note: `${concat($field, _MASK)}` preserves `$field`'s case. To get
upper-case, use `${concat($field:upper, _MASK)}` (once
`macro_metavar_expr_concat` supports case-conversion fragments) or
require consumers to name fields in the desired case. For today
we generate `<field>_MASK` (lower-case field name), e.g.
`origin_MASK`. If the convention should be `ORIGIN_MASK` we either
(a) wait on fragment-case-conversion, or (b) document the
lower-case convention and accept it.

### Concrete decision

Emit `<field>_MASK` (lower-case field prefix). The `_MASK` suffix is
upper-case to flag it as a constant; the field-name portion matches
the field's spelling for direct-lookup mental model
(`StrLayout::origin_MASK` pairs with `.origin()` accessor).

## `mock/crates/arvo-bits/tests/bitfield.rs`

Add assertions on the generated mask constants:

```rust
#[test]
fn masks_match_layout() {
    assert_eq!(StrHandle::origin_MASK.bits(), 1u64 << 31);
    assert_eq!(StrHandle::reserved_MASK.bits(), 0b111u64 << 28);
    assert_eq!(StrHandle::id_MASK.bits(), 0x0FFF_FFFF);
}

#[test]
fn from_u64_matches_new() {
    let via_new = Bits::<32, Hot>::new(0x1234);
    let via_from: Bits<32, Hot> = 0x1234u64.into();
    assert_eq!(via_new, via_from);
}

#[test]
fn from_container_matches_from_raw() {
    let via_raw = Bits::<8, Hot>::from_raw(0x42u8);
    let via_from: Bits<8, Hot> = 0x42u8.into();
    assert_eq!(via_raw, via_from);
}
```
