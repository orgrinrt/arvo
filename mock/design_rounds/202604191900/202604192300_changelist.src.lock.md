### Impl-time decisions (pre-committed)

All six decisions carry forward identically from 202604192100 / 202604192200. The `Warm` compile-time error at `I + F > 32` is implemented structurally: the sealed `UContainerFor<BITS>` and `IContainerFor<BITS>` traits have impls for `Hot` / `Cold` / `Precise` across `BITS = 1..=64` but intentionally omit `Warm` for `BITS > 32`. An unreachable `WarmCompatible` sealed trait is available as a fallback encoding if const-expression bounds ever need to reference it explicitly. `Strategy::Container<const I: IBits, const F: FBits>` remains the conceptual shape; the concrete mechanism is the bridge-trait-keyed-by-BITS pattern (`UArith<BITS>` / `IArith<BITS>` supertrait `UContainerFor<BITS>` / `IContainerFor<BITS>`) which avoids nightly const-expression cycles. Every numeric wrapper is `repr(transparent)`. `From` is provided for widen-free strategy conversions (`Hot → Warm`, `Hot → Precise`, `Warm → Precise`); `TryFrom` for narrowing (with logical-range bounds checks). The per-width float options considered and rejected during the 202604192000 open-question phase (`StrictFloat32`, `StrictFloat64`, and the symmetric `FastFloat32` / `FastFloat64` pair) are superseded by the generic-over-`Ieee` encoding; their names remain here for comparison-lint coverage only.

Cross-width arithmetic (`Add<UFixed<I2, F2, S2>> for UFixed<I1, F1, S1>` with `I = max(I1, I2)`, `F = max(F1, F2)`, `S = <S1 as Resolve<S2>>::Out`) is deferred with in-code TODO comments on both `UFixed` and `IFixed` — blocked on `generic_const_exprs` support for `max()` in const arguments on current nightly. Same-strategy same-width arithmetic is the shipping baseline. The deferral is recorded in `BACKLOG.md.tmpl` and carried from all three deprecated drafts.

## Impl-time decisions (pre-committed)

1. **Float typing.** Sealed `Ieee: sealed::Sealed + Copy + Default + PartialEq + PartialOrd + 'static` with `WIDTH: u8` / `ZERO: Self` / `ONE: Self`. Impl'd for `f32` and `f64`. `FastFloat<F: Ieee>(F)`, `StrictFloat<F: Ieee>(F)`, `Float<F: Ieee = f32>` cfg-resolved.
2. **Warm const-bound.** Enforced by omission of `UContainerFor<BITS>` / `IContainerFor<BITS>` impls on `Warm` for `BITS > 32`. Missing impl → type fails to resolve → compile error at use site.
3. **Container types.** Sealed `UContainerFor<const BITS: u8>` and `IContainerFor<const BITS: u8>` traits keyed by logical bit count (`I + F` for UFixed, `1 + I + F` for IFixed). Macro-expanded impls cover every valid `(strategy, BITS)` pair.
4. **`repr(transparent)`** on every numeric wrapper. `size_of` verified in tests.
5. **Strategy conversion.** `From` for widen-free paths (Hot→Warm, Hot→Precise, Warm→Precise). `TryFrom` for narrowing (Warm→Hot, Precise→Hot) with logical-range bounds checks using `u128`/`i128` intermediaries.
6. **Cross-width ops.** Deferred with explicit TODO comments. Same-strategy same-width is the shipping baseline.
