### 5. `mock/crates/arvo-comb/` — signature + internal rewrites

- **`Cargo.toml`** — add `arvo-tensor.workspace = true`. Keep `arvo-bitmask.workspace = true` (re-exports may still be used transitively in tests).
- **`src/lib.rs`** — re-exports remain (`bin_pack`, `matrix_chain_dp`, `greedy_group`, `Range`). Import `arvo_tensor::{Array, Matrix, cap_size}` at crate root for use across modules.
- **`src/binpack.rs`** — rewrite:
  - Input: `items: &Array<T, N>` (was `&[T; cap_size(N)]`).
  - Output: `(USize, Array<USize, N>)` (was `(USize, [USize; cap_size(N)])`).
  - Internal indices: raw `usize` loop variables replaced by either `for i in 0..cap_size(N)` (inferred usize) or `USize` wrapping where a variable holds a logical index across multiple lines. Choose on a per-loop basis for readability.
  - Internal `score: [W; N]`, `order: [USize; N]`, `used: [W; B]` — keep as raw arrays (single-method-local scratch; no API visibility).
  - `placed: bool` flag → `placed: Bool`.
- **`src/dp.rs`** — rewrite:
  - Callback sig: `feasible: impl Fn(USize, USize) -> Bool`.
  - DP tables: `dp: Matrix<W, N>`, `reachable: Matrix<Bool, N>`, `split: Matrix<USize, N>`.
  - Output: `(W, Array<USize, N>)`.
  - `fill_splits` helper — take `&Matrix<USize, N>` / `&Matrix<Bool, N>` / `&mut Array<USize, N>`.
  - `best_set: bool` → `Bool`.
- **`src/greedy.rs`** — rewrite:
  - Input: `items: &Array<T, N>`.
  - Callback sig: `feasible: impl Fn(&A, &T) -> Bool`.
  - Output: `(USize, Array<Range, M>)`.
  - Local `open: bool` → `Bool`.
- **`tests/*.rs`** — update closure returns from `bool` to `Bool`; convert `[T; N]` literals to `Array::<T, N>::from_fn(...)` or `Array::new([...])`.
