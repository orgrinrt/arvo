## Files

### 1. `arvo-bits/src/traits.rs` — add `BitLogic` trait

Append after `BitSequence`:

```rust
/// Whole-word bitwise logic.
///
/// Single-instruction on the backing primitive (x86 `or` / `and` /
/// `xor` / `not`; identical on aarch64). Loop-free. The three
/// bit-level contracts partition: `BitAccess` covers SINGLE-BIT
/// read/write; `BitSequence` covers SCAN / POPCOUNT; `BitLogic`
/// covers WHOLE-WORD logical ops. Together they describe a
/// bit-bearing container.
///
/// Hot-only surface. Other strategies (`Warm`, `Precise`, `Cold`)
/// have wider containers that make whole-word ops meaningless at the
/// logical-width level.
pub trait BitLogic: BitWidth + Copy {
    /// Whole-word OR.
    fn bitor(self, other: Self) -> Self;
    /// Whole-word AND.
    fn bitand(self, other: Self) -> Self;
    /// Whole-word NOT.
    fn bitnot(self) -> Self;
    /// Whole-word XOR.
    fn bitxor(self, other: Self) -> Self;

    /// Clear the lowest set bit.
    ///
    /// Default impl uses `BitSequence::trailing_zeros` +
    /// `BitAccess::with_bit_cleared`. Concrete types can override
    /// with `self.to_raw() & (self.to_raw() - 1)` for the single-
    /// instruction fast path (x86 `BLSR`). Returns `self` unchanged
    /// when the word is already zero.
    fn clear_lowest_set_bit(self) -> Self
    where
        Self: BitAccess + BitSequence,
    {
        if <Self as BitSequence>::is_zero(self).into() {
            return self;
        }
        let idx = <Self as BitSequence>::trailing_zeros(self);
        <Self as BitAccess>::with_bit_cleared(self, idx)
    }
}
```

### 2. `arvo-bits/src/ufixed_impl.rs` — concrete `BitLogic` + `core::ops` impls

nicer but isn't load-bearing.

Rewrite `impl Mask<QWord<Hot>>` block:
- `union` → `<QWord<Hot> as BitLogic>::bitor(self.word, other.word)` or
  `self.word | other.word` via `core::ops::BitOr`.
- `intersection` → `BitLogic::bitand` equivalent.
- `difference` → `BitLogic::bitand(self.word, BitLogic::bitnot(other.word))`.
- `complement` → `BitLogic::bitnot`.
- `is_empty` → already uses `BitSequence::is_zero`; no change.
- `intersects` → `!<QWord<Hot> as BitSequence>::is_zero(BitLogic::bitand(self.word, other.word))`. Use `Bool`'s `!` implementation (already defined) or compute via `!Bool::into`.
- `contains` / `insert` / `remove` → already use `BitAccess`; no change.
- `count` → already uses `BitSequence::count_ones`; no change.
- `lowest_set` → already uses `BitSequence::trailing_zeros`; no change.
- `highest_set` → currently uses `qword_to_u64(...).leading_zeros() as usize`.
  Rewrite to use `BitSequence::leading_zeros` (returns `USize`) +
  `BitSequence::is_zero`. Empty-word path returns `USize(64)`. Non-empty:
  `USize(63) - leading_zeros` in USize arithmetic. Need USize::sub or use
  `.0` access + reconstruct.

Rewrite `SetBitsIter64`:

```rust
/// Nameable iterator over set bits of a `Mask64`.
#[derive(Copy, Clone)]
pub struct SetBitsIter64 {
    remaining: QWord<Hot>,
}

// lint:allow(no-bare-option) reason: core::iter::Iterator::next trait-method signature returns Option<Self::Item>; tracked: #115
impl Iterator for SetBitsIter64 {
    type Item = USize;

    #[inline(always)]
    fn next(&mut self) -> Option<USize> {
        if <QWord<Hot> as BitSequence>::is_zero(self.remaining).into() {
            return None;
        }
        let idx = <QWord<Hot> as BitSequence>::trailing_zeros(self.remaining);
        self.remaining = <QWord<Hot> as BitLogic>::clear_lowest_set_bit(self.remaining);
        Some(idx)
    }
}
```

Rewrite `SetBitsIter256`:

```rust
/// Nameable iterator over set bits of a `Mask256`.
#[derive(Copy, Clone)]
pub struct SetBitsIter256 {
    words: [QWord<Hot>; 4],
    word_idx: USize,
}

// lint:allow(no-bare-option) reason: core::iter::Iterator::next trait-method signature returns Option<Self::Item>; tracked: #115
impl Iterator for SetBitsIter256 {
    type Item = USize;

    #[inline(always)]
    fn next(&mut self) -> Option<USize> {
        while self.word_idx.0 < 4 {
            let w = self.words[self.word_idx.0];
            if !<QWord<Hot> as BitSequence>::is_zero(w).into() {
                let bit = <QWord<Hot> as BitSequence>::trailing_zeros(w);
                self.words[self.word_idx.0] = <QWord<Hot> as BitLogic>::clear_lowest_set_bit(w);
                return Some(USize(self.word_idx.0 * 64 + bit.0));
            }
            self.word_idx = USize(self.word_idx.0 + 1);
        }
        None
    }
}
```

Rewrite `iter_set_bits` on `Mask256` similarly.
