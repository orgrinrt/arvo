## Decisions (mechanical; no new design)

**All six sites** are local mutable state — counters and flag arrays, not structural types. Resolution pattern is already established by rounds 1-2:

- `let mut x: usize = 0;` → `let mut x = USize(0);` with `.0` to extract for arithmetic where `usize`-typed comparisons occur inline.
- `let mut flag: bool = false;` → `let mut flag = Bool::FALSE;` with `.0` extraction.
- `[bool; cap_size(N)]` internal flag arrays → `[Bool; cap_size(N)]` with `[Bool::FALSE; cap_size(N)]` init.

No structural wrapper needed — these are not public signatures, not arrays that cross boundaries. The `arvo-tensor::Array<Bool, N>` wrapper is overkill for `is_waist`; staying on raw `[Bool; cap_size(N)]` is the minimum delta.

Other patterns surfaced but already clean in this crate:
- Loop indices `while i < cap_size(N); i += 1;` already rewritten as `for i in 0..cap_size(N)` in prior rounds of this crate.
